# Credit Card Customer Churn Analysis

## 📌 Project Overview
This project analyzes credit card customers to understand churn behavior.

## 🎯 Objective
Identify key factors that lead to customer attrition.

## 📊 Key Insights
- Low transaction activity leads to higher churn
- Inactive customers are more likely to leave
- Age has no significant impact

## 🛠️ Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib

## 📁 Dataset
Kaggle Credit Card Customers Dataset
